# 📌 Student Management System

## 🚀 Features

- Admin login with password hashing
- Session-based authentication
- Add, View, Edit, Delete students (CRUD)
- MySQL database integration
- Secure password storage using Werkzeug

## 🛠 Tech Stack

- Python
- Flask
- MySQL
- PyMySQL
- HTML (Jinja templates)

## 🏗 Architecture

Browser → Flask → MySQL Database

## ▶ Run Locally

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
